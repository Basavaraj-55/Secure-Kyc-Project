# 🔐 Secure and Transparent KYC for Banking System Using IPFS & Blockchain

A secure blockchain-based KYC verification system that enables banks and financial institutions to verify customer identities efficiently using **IPFS (InterPlanetary File System)** and **Blockchain Technology**. The system eliminates repeated KYC verification, improves security, reduces costs, and ensures transparency.

---

# 🚀 Project Overview

Traditional Know Your Customer (KYC) verification requires customers to repeatedly submit identity documents whenever they open an account with a different bank. This process is time-consuming, costly, and vulnerable to document tampering.

This project leverages **Blockchain Technology** and the **InterPlanetary File System (IPFS)** to create a secure, transparent, and decentralized KYC verification platform. Once a customer completes the KYC process with one bank, the encrypted documents are stored securely on IPFS while their hash is recorded on the blockchain. Other authorized banks can verify the customer's identity without requesting the documents again, making the entire process faster, safer, and more efficient.

# 🔄 System Workflow

1. Customer registers on the platform.
2. Customer uploads KYC documents.
3. Documents are encrypted.
4. Encrypted files are uploaded to IPFS.
5. IPFS generates a unique hash.
6. Hash is securely stored on Blockchain.
7. Customer receives a private key.
8. Another bank requests verification.
9. Blockchain verifies the document hash.
10. Authorized bank retrieves encrypted files from IPFS.
11. Customer identity is verified successfully.
